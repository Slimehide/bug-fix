About Island Game Studio

Island Game Studio is video game studio that creates PC & console games. We make games that are fun to play, visually distinct and built to be played again and again.

For us, games are a way to invite others into the worlds we create. Giving everyone a chance to discover something new and exciting. It’s a lot like reggae, once you find the rhythm, you follow it.

Island Game Studio is a small-scale indie games studio passionate about crafting games that ignite joy.
