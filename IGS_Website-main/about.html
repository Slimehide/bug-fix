About Island Game Studio

Island Game Studio is a video game studio with Caribbean roots that specializes in the development of PC & console games. We create Caribbean inspired games for a global audience.
Our games are visually distinct, highly repayable and above all else, fun. Everyone deserves a relaxing getaway.

Island Games is built around a simple idea: small teams can create wildly imaginative systems if they focus on depth instead of breadth.
